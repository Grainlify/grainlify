# DataPage Component Update

## Overview
This update improves the `DataPage.tsx` component layout and styling for better readability, usability, and visual presentation.

### Key Changes:
- **Reorganized Layout:**  
  - Project Activity and Contributor Activity stacked on the left column.  
  - Contributors Map on the right column.  
  - Information panel at the bottom.
- **Chart Adjustments:**  
  - Reduced chart heights for better fit.  
  - Added gradients and smoother transitions.
- **Filters & Buttons:**  
  - Added toggle functionality for project and contributor filters.  
  - Styled buttons with hover and active states for clarity.
- **Map Enhancements:**  
  - Zoom controls added.  
  - Highlighted specific countries with markers and connection lines.
- **Theme Support:**  
  - Works with dark and light themes using `useTheme` context.
